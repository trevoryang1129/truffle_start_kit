# truffle_start_kit
truffle 项目基础框架，web 客户端使用quasar,vue,typescript,web3
网上看到大部分环境搭建教程都是把web 工程与 Solidity 工程放在同一目录中，但对于大型项目，web 开发人员与区块链开发工程师分属两个不同岗位，因此有必要分开目录，因为这里没有使用Go 等后端语言做相关开发，故在此只分开2个目录。一个是web 前端，另外一个是solidity 工程目录。
